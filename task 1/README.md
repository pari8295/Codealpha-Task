## 🌍 Language Translator Tool

A simple and user-friendly **Language Translation Tool** built using **HTML, CSS, and JavaScript**. This application allows users to translate text between multiple languages using a translation API. It provides an intuitive interface for entering text, selecting source and target languages, and viewing translated results instantly.

### ✨ Features

* 🌐 Translate text between multiple languages
* 🔄 Select source and target languages from dropdown menus
* ⚡ Real-time translation using a Translation API
* 📋 Copy translated text with a single click
* 🎨 Clean and responsive user interface
* 📱 Works on desktop and mobile devices
* 🔊 Optional Text-to-Speech support for translated text

### 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Translation API (Google Translate API / MyMemory API)

### 🚀 How It Works

1. Enter text in the input box.
2. Select the source language.
3. Select the target language.
4. Click the **Translate** button.
5. The translated text is displayed instantly.
6. Use the **Copy** button to copy the translated result.

### 📸 Project Preview

This project demonstrates the integration of external translation APIs with a responsive front-end interface, making language conversion fast and accessible for users.
